# Browser Tools

A privacy-first browser toolkit built with Flask.

## Run locally

```powershell
python -m venv .venv
.venv\Scripts\Activate.ps1
pip install -r requirements.txt
python app.py
```

Open `http://127.0.0.1:5000` in your browser.

## Production deployment

The repository includes a Render Blueprint in `render.yaml` and uses Python 3.13 with Gunicorn in production.

- Build command: `pip install -r requirements.txt`
- Start command: `gunicorn --bind 0.0.0.0:$PORT --workers 2 --threads 4 --timeout 120 app:app`
- Health check: `/healthz`

Before a public launch, configure these Render environment values:

- `SITE_URL`: the final origin, such as `https://example.com`, with no trailing slash
- `CONTACT_EMAIL`: the public support address shown on the Contact page

`robots.txt`, `sitemap.xml`, canonical URLs, production security headers, and custom error pages are generated by Flask.

## Features

- Client-side PDF merging with no file uploads
- PDF ordering and removal controls
- Visual PDF splitting with page previews and selectable cut points
- ZIP downloads for split PDF results
- PDF processing in a browser worker
- Wide path editor with selectable paths, visible start markers, color-coded endpoint hover, automatic endpoint joining, Alt-drag Bezier handles, image layers, opacity, and layer ordering
- Downloadable project files that can be reopened to continue editing
- Multi calculator with arithmetic, percent-of, percentage-change, discount, sales-tax, tip, and split-bill modes
- Detailed About, tool guides, FAQ, Privacy Policy, Terms, and Contact pages
- International discovery pages in English, Korean, Japanese, Spanish, French, German, Portuguese, Simplified Chinese, Hindi, and Arabic, with reciprocal hreflang metadata
- 5, 15, 25, and 45-minute presets
- Custom durations from 1 to 180 minutes
- Start, pause, and reset controls
- Completion sound
- Timer state persists after a refresh
- Keyboard shortcuts: Space (start/pause), R (reset)
